To initialize the database, run:

```
python initialize_db.py
```


Configure the venv

```
python -m venv venv
source venv/bin/activate  # On Linux/macOS

pip install sqlmodel
pip install matplotlib
pip install pyqt5
```

Alternatives

```
python -m venv venv
source venv/bin/activate  # On Linux/macOS

pip install -r requirements.txt
```
